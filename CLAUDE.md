# 정책자금 컨설팅 회사 내부 전산시스템

## 한 줄 요약
영업팀(고객관리·계약)과 관리팀(자금신청 진행)의 업무 흐름을 자동화하는 웹앱. 계약 완료 → 자동으로 관리팀에 신규업체로 인수인계.

## 기술 스택 (고정)
- **프론트/서버**: Next.js (App Router)
- **코드 보관**: GitHub
- **배포**: Vercel
- **DB**: Supabase (PostgreSQL)
- **인증**: Supabase Auth
- **이메일**: Resend
- **파일 저장**: Supabase Storage

## 핵심 원칙
1. **1단계는 돌아가는 최소 버전**만 (더미 데이터, 레이아웃만)
2. **욕심 금지**: MVP에 없는 건 v2로 미룸 (성과 대시보드, 마감일 알림 등)
3. **플랜 먼저**: 파일 수정 전에 항상 로드맵 검토
4. **권한 먼저**: 데이터 먼저 만들고, 권한은 나중에 추가

## 개발 순서 개요
1. Next.js 레이아웃 + 더미 데이터 ← **오늘 (지금)**
2. Supabase 연동 (DB + CRUD API)
3. 권한 관리 (NextAuth.js)
4. 자동화 트리거 (계약완료 → 신규업체 생성)
5. 알림 시스템 (이메일 + 대시보드)
6. 칸반 보드 고도화
7. UI/UX 다듬기
8. 배포 & 운영 준비
